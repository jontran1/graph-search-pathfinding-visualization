Time Complexity: O(b^d)

Space Complexity: O(b^d)

Breadth-first search is an algoirthm for traversing or searching a graph or tree structure. It is a blind search (uninformed) menaing the algorithm searches blindly through out the graph until the target node is found. It uses the opposite strategy of depth-first search. Meaning, it explores the nodes level by level or in this case for the maze, in a circular fashion.

When would you use this?: If for some magical reason you believe the target node to be close to the root, BFS first search would be a good choice. This algorithm is complete, so eventually BFS will find the target node. However, space complexity becomes a problem because each node is requried in memory for the search to be perform.
